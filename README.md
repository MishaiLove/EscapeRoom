# EscapeRoom

Консольная игра “Escape Room”.

## Цель
Собрать ключ `?`, открыть дверь (`;` → `:`) и выйти из комнаты через открытую дверь.

## Управление
- Стрелки: движение на 1 клетку
- ESC: выход из игры

## Обозначения
- `#` — стена
- `.` — пол
- `!` — игрок
- `?` — ключ
- `;` — закрытая дверь
- `:` — открытая дверь
